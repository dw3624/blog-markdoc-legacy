# 진수

### 10진수 → 타 진수로 변환

- 원하는 타진법의 수로 나눈 뒤 나머지를 거꾸로 읽는다



### 2진수, 8진수, 16진수간 변환

- 2진법 → 8진법 : 3자리씩 묶음
- 8진법 → 2진법 : 3자리씩 나열
- 2진법 → 16진법 : 4자리씩 묶음
- 16진법 → 2진법 : 4자리씩 나열





# 실수

### 실수의 표현

- 컴퓨터는 실수를 표현하기 위해 부동 소수점(floating-point) 표기법을 사용
- 부동 소수점 표기 : 소수점의 위치를 고정시켜 표현하는 방식
