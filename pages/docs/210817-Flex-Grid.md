# Flex와 Grid의 차이

- Flex - 한 방향 레이아웃 시스템 (1차원)
- Grid - 두 방향 (가로-세로) 레이아웃 시스템 (2차원)


# bootstrap에서의 사용법

## Flex

```html
<div class="d-flex p-2 bd-highlight">
    I'm a flexbox container!
</div>
<div class="d-inline-flex p-2 bd-highlight">
    I'm an inline flexbox container!
</div>
```



## Grid System

```html
<div class="container">
    <div class="row">
        <div class="col-sm">
            One of three columns
        </div>
        <div class="col-sm">
            One of three columns
        </div>
        <div class="col-sm">
            One of three columns
        </div>
    </div>
</div>
```
